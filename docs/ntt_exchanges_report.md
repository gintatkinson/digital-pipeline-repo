# NTT Telecom Exchanges and Offices in Japan

This report contains the clean, filtered dataset of **763 NTT-related locations** (exchanges, offices, hospitals, customer centers) mapped in OpenStreetMap across Japan.

## Data Analysis & Tagging Context
* **Query Source:** OpenStreetMap nodes matching `"name" ~ "NTT|ＮＴＴ|日本電信電話"`.
* **Clean Filtering:** Stands alone from generic Japanese matching terms (like standalone "東日本" or "西日本" which are found in non-telecom names like railways or banks) to isolate active telecommunications facilities.
* **Tagging Consistency:** In OpenStreetMap, many NTT facilities are mapped as generic buildings (`building=yes`) or offices rather than technical exchanges (`telecom=exchange`). This list captures both for maximum coverage.
